Music API

The goal for this group project was to create an API that made use of a database that we had designed and created by ourselves and also incorporate the functionality of an external API.

This API stores information on artist names, genre names and song names. Each user can "like" and store a selection of different artists, genres and songs and each of these datatypes also had connections with one another.

A simple console program was created to navigate through the data found in the API.
